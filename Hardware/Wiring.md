# 🔌 ESP32 Wiring Connections

| Component | ESP32 Pin |
|-----------|-----------|
| DHT11 DATA | GPIO26 |
| Relay IN | GPIO27 |
| Rain Sensor | GPIO25 |
| Soil Moisture AO | GPIO34 |
| LCD SDA | GPIO33 |
| LCD SCL | GPIO32 |

## Power Connections

- ESP32 3.3V → DHT11 VCC
- ESP32 GND → All GND Pins
- Relay VCC → 5V
- LCD VCC → 5V
